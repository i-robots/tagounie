# Tagounie Web (Vue + Vite)

This repository was converted from a static site into a small Vue 3 + Vite project so you can run, develop, and build it.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

Notes
- The existing CSS remains at `style/index.css` and is imported from `src/main.js`.
- Images and video files remain in the `images/` folder; paths are unchanged.
